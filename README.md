## 🚀 Overview

This project demonstrates how to build **AI Guardrails** for LLM-powered agents.

The notebooks showcase different safety mechanisms including:

- 🔒 **PII Detection & Masking**
- 🧑‍⚖️ **Human-in-the-Loop Approval**
- 🚫 **Deterministic Input Guardrails**
- 🤖 **Tool-based AI Agents**
- 🧠 **Local LLMs using Ollama**

The goal is to understand how to **secure AI systems before production deployment**.
